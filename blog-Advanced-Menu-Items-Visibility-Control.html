Version v1.2.3

30 décembre 2025

La version 1.2.3 du projet Advanced-Menu-Items-Visibility-Control corrige un problème critique lié à la détection du niveau d'adhésion RCP, qui renvoyait des données obsolètes issues du cache. Une requête directe sur les adhésions a été mise en place afin de contourner le bug de mise en cache présent dans la fonction rcp_get_customer_membership_level_names() de l'API RCP. Un contrôle via function_exists() a été ajouté pour la fonction rcp_user_has_access_level(), afin d'éviter des erreurs fatales sur certaines versions de RCP. Enfin, la comparaison des niveaux d'adhésion a été modifiée pour reposer sur les noms plutôt que sur les identifiants, dans le but d'améliorer la fiabilité de la correspondance.
