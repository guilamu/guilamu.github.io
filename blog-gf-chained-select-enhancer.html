Version v1.8.0

24 février 2026

La version 1.8.0 de gf-chained-select-enhancer apporte exclusivement des correctifs de sécurité. Une vulnérabilité XXE (XML External Entity) présente dans le parseur XLSX a été corrigée par l'ajout du drapeau LIBXML_NONET et une protection du chargeur d'entités. Le gestionnaire d'upload XLSX intègre désormais une vérification de capacité via current_user_can, le nonce seul ne constituant pas un contrôle d'autorisation suffisant. Deux vecteurs de type XSS ont été éliminés : l'un en sortie CSS, désormais assainie via wp_strip_all_tags(), l'autre dans le JavaScript d'administration où les manipulations innerHTML ont été remplacées par des méthodes DOM sécurisées. Les appels aux fonctions internes wp_kses_hook() et wp_kses_split() ont été remplacés par l'API publique wp_kses(). L'en-tête Content-Type: application/json est maintenant explicitement envoyé avec les réponses JSON de l'upload XLSX. L'export CSV bénéficie de l'assainissement du nom de fichier via sanitize_file_name() ainsi que de l'en-tête X-Content-Type-Options: nosniff. Une validation MIME des fichiers XLSX a été ajoutée, vérifiant la structure ZIP avant tout traitement. Enfin, le nonce et l'URL AJAX ne sont plus exposés que sur les pages de l'éditeur Gravity Forms.
