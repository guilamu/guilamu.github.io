Version v1.7.2

5 février 2026

La version 1.7.2 de gf-chained-select-enhancer corrige trois anomalies. La compatibilité avec PHP 7.4 est rétablie dans le module de mise à jour GitHub : la fonction str_ends_with(), disponible uniquement à partir de PHP 8, a été remplacée par un appel à substr(). Une erreur de décalage d'indice dans la boucle de la fonction array_values_recursive(), utilisée lors de l'import de fichiers XLSX, a également été corrigée. Enfin, l'incohérence de licence entre l'en-tête du plugin et le fichier README a été résolue : la licence AGPL-3.0 est désormais indiquée uniformément dans les deux emplacements.

Version v1.4.0

31 décembre 2025

La version 1.4.0 du projet gf-chained-select-enhancer introduit une architecture modulaire reposant sur des fichiers de classes séparés, ainsi qu'une externalisation des ressources CSS et JavaScript. La sécurité a été renforcée par l'ajout d'une gestion appropriée des nonces et une sanitisation des données. Les performances ont été améliorées grâce au chargement conditionnel du CSS, limité aux pages qui affichent effectivement des formulaires, et par la mise en cache des appels à l'API GitHub via un transient d'une durée de douze heures. L'export CSV a par ailleurs été corrigé afin de diffuser les données en flux direct, supprimant ainsi tout stockage de fichiers dans l'espace public du serveur.

Version v1.3

26 novembre 2025

La version 1.3 du plugin gf-chained-select-enhancer, publiée le 26 novembre 2025, introduit un système de mises à jour automatiques depuis le dépôt GitHub. Le plugin est désormais capable de détecter et d'installer les nouvelles versions sans intervention manuelle, grâce à l'ajout d'un champ Update URI dans l'en-tête du plugin, qui permet l'intégration native avec le mécanisme de mise à jour de WordPress.
