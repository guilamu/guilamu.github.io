Version v1.1.0

21 février 2026

La version 1.1.0 de gravity-extract introduit la prise en charge de l'API Google Gemini comme fournisseur d'intelligence artificielle alternatif à POE, avec la possibilité de sélectionner le fournisseur individuellement par formulaire. Chaque formulaire peut désormais disposer de sa propre clé API Gemini ainsi que du choix du modèle à utiliser, les modèles Gemini Flash et Gemini Pro étant tous deux supportés. Le routage des appels API a été mis à jour pour tenir compte du fournisseur sélectionné sur l'ensemble des fonctionnalités d'extraction, à savoir l'analyse, l'automapping et la détection. La documentation README a également été mise à jour pour couvrir cette configuration multi-fournisseurs.
