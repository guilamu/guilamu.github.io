Signalement unifié de bugs pour plugins Guilamu avec réponses IA instantanées.

Version v1.2.1

19 janvier 2026

La version 1.2.1 de guilamu-bug-reporter apporte plusieurs modifications. Sur le plan de la sécurité, le token GitHub utilisé a été remplacé par un token à granularité fine, dont les permissions sont désormais restreintes aux seuls dépôts guilamu. Une extraction intelligente du contenu README a été ajoutée, permettant de récupérer automatiquement les sections FAQ, prérequis et problèmes connus afin de fournir un contexte documentaire à l'intelligence artificielle. En conséquence, les réponses générées par l'IA s'appuient maintenant sur cette documentation du plugin pour proposer un diagnostic et un dépannage plus pertinents.

Version v1.1.2

9 janvier 2026

La version 1.1.2 du projet guilamu-bug-reporter apporte plusieurs ajustements à l'interface utilisateur : le titre de succès en vert a été supprimé afin d'obtenir une interface plus épurée, la zone d'affichage de la réponse de l'IA a été agrandie grâce à une réduction des marges internes, et la disposition des boutons a été revue avec le bouton de fermeture repositionné à gauche et le lien vers GitHub présenté sous forme de bouton bleu à droite.
